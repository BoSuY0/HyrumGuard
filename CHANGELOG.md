# Changelog

## 1.0.0 - Stable release candidate

- Establishes HyrumGuard as a local CLI/package release candidate.
- Supports downstream discovery through manual seeds and ecosyste.ms package dependent lookup.
- Mines Python and JavaScript/TypeScript downstream usage into shadow contracts.
- Reports PR-time implicit compatibility risks as JSON, Markdown, and SARIF.
- Plans changed-only canary checks by default and requires explicit opt-in for execution.
- Adds artifact validation, release documentation, and release quality gates.
